# 🚀 🧾 COMMANDS (STEP BY STEP)

👉 Run all commands from your project root (`Re-creation/`)

---

## 🔍 1. Dataset Analysis (Sanity Check)

```bash
python -m dataset_analyzer.analyze_dataset
```

👉 Confirms:

* paths correct
* masks exist
* dataset integrity

---

## 🫁 2. Train Lung Model

```bash
python -m training.train_lung
```

✔ Output:

* `checkpoints/lung_model.pth`
* `logs/lung_training_log.csv`
* `logs/lung_metrics.png`

---

## 🦠 3. Train Infection Model

```bash
python -m training.train_infection
```

✔ Output:

* `checkpoints/infection_model.pth`
* `logs/infection_training_log.csv`
* `logs/infection_metrics.png`

---

## 📊 4. Evaluate Lung Model

```bash
python -m evaluation.evaluate
```

✔ Output:

* Dice score printed

---

## 🧪 5. Ablation Study

```bash
python -m evaluation.ablation
```

✔ Output:

* Segmentation vs severity comparison

---

## 🆚 6. Comparison Graph (VERY IMPORTANT)

```bash
python -m evaluation.compare_models
```

✔ Output:

```text
logs/comparison_dice.png
logs/comparison_iou.png
```

---

## 🧠 7. Run Full Pipeline (Single Image)

```bash
python -m pipeline.run_pipeline
```

✔ Output:

* segmentation overlay
* Grad-CAM
* severity

---

## 🌐 8. Run Streamlit App

```bash
streamlit run app/app.py
```

👉 Opens browser UI
